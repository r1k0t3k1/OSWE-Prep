External Commands

In some cases you may need to send an external command to the monitoring engine for transmitting a passive service state or something similiar.
This interface was designed to help you create these commands by generating an example for each supported command. Additionally you see a list of supported parameters and default values
API structure:
hostUuid
required
sticky
0
notify
1
persistent
1
author
required
comment
required

Example HTTP request using default parameters:
https://192.168.217.129/nagios_module/cmd/submit/command:ACKNOWLEDGE_HOST_PROBLEM/hostUuid:$required/author:$required/comment:$required/.json?apikey=USER_API_KEY
